# Zikaron - A Social Music Game

Zikaron is a simple web-based social music game where a word is displayed on the screen, and players have to find a song with that word in it.

## How to Play

1.  Open the `index.html` file in a web browser.
2.  A word will be displayed on the screen.
3.  Find a song with that word in it.
4.  Click the "New Word" button to get a new word.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The stylesheet for the application.
*   `script.js`: The JavaScript file containing the game logic.
*   `assets/logo.svg`: The logo for the application.
